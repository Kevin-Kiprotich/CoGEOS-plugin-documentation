Installation

Package Installation

The CoGEOS Shoreline Extraction plugin works with external python libraries when handling most of its analysis. Therefore, before the plugn is installed, you would have to install all external libraries with the following procedure.

  1. Open the QGIS python console.

  2. Click on show editor .

  3. Paste the following code into the untitled file.

  4. Click run.

 1import subprocess
 2
 3packages=['earthengine-api','folium', 'rasterio', 'rtree', 'pandas', 'geopandas', 'pyCRS', 'geemap', 'mapclassify', 'contextily', 'matplotlib_scalebar', 'opencv-python', 'natsort', 'scikit-learn']
 4try: 
 5    for package in packages:
 6        subprocess.run(["pip","install", package])
 7        print(f"{package} installed successfully")
 8
 9except subprocess.CalledProcessError as e:
10    print(f"Error installing the dependencies: {e}")

Install the Plugin

To install the plugin, follow the following procedure.

  1. Download the plugin zip here.

  2. Open the QGIS plugin manager

  3. Click install from ZIP.

  1. Browse to the download directory.

  2. Install the plugin.
